1. Build a Simple Blockchain
## Goal: Create a basic blockchain from scratch to understand its core components.

### What You'll Learn:
Data structures like blocks, chains, and Merkle trees.
Cryptographic hashing (e.g., SHA-256).
Proof of Work (PoW) or Proof of Stake (PoS) consensus.
Transaction validation and block mining.
Tech Stack:
Programming Language: Python, JavaScript, or C++.
Libraries: hashlib (Python), crypto (JavaScript).
Outcome:
A simple command-line blockchain where users can mine blocks, add transactions, and view the chain.
